4.1 Word Processing

[span_34](start_span)

Word processing software (like Microsoft Word or Google Docs) is used to create, edit, format, and print text documents[span_34](end_span).

Key Features and Concepts

    [span_35](start_span)
  • Basics & Features: Typing text, saving files, and using templates[span_35](end_span).
    • [span_36](start_span)
  • Clipboard: A temporary storage area used for Cut (remove and copy), Copy, and Paste operations[span_36](end_span).
    • [span_37](start_span)
  • Font Formatting: Changing the appearance of text (e.g., bold, italic, underline, font size, font color)[span_37](end_span).
  • Page and Paragraph Formatting:
      [span_38](start_span)
    • Paragraph: Adjusting alignment (left, center, right), line spacing, and indentation[span_38](end_span).
      • [span_39](start_span)
    • Page: Setting margins, page orientation (portrait/landscape), and page size[span_39](end_span).
    • [span_40](start_span)
  • Table Creation: Inserting a grid of rows and columns to organize data[span_40](end_span).
    • [span_41](start_span)
  • Page Setup: The process of defining the layout of the document (margins, orientation, etc.)[span_41](end_span).
    • [span_42](start_span)
  • Spelling and Grammar: Built-in tools that check for and suggest corrections for spelling and grammatical errors[span_42](end_span).

4.2 Spreadsheet

[span_43](start_span)

Spreadsheet software (like Microsoft Excel or Google Sheets) is used to organize, analyze, and visualize data in a grid of rows and columns[span_43](end_span).

Key Features and Concepts

    [span_44](start_span)
  • Spreadsheet Concept: A digital ledger consisting of a Workbook (the file) which contains one or more Worksheets (the grids)[span_44](end_span).
  • Elements of Spreadsheet:
    • Cell: The intersection of a row and a column.
      • [span_45](start_span)
    • Cell Address: The unique name of a cell, given by its column letter and row number (e.g., A1, B2)[span_45](end_span).
      • [span_46](start_span)
    • Formula Bar: An input bar used to enter or edit data and formulas in the active cell[span_46](end_span).
  • Formulas: Used to perform calculations. Formulas always start with an = sign. Examples:
    • =A1+B1 (Adds the values in A1 and B1).
      • [span_47](start_span)
    • =SUM(C1:C10) (Adds all values in the range C1 to C10)[span_47](end_span).
    • [span_48](start_span)
  • Chart: A graphical representation of data (e.g., Pie Chart, Bar Chart, Line Chart)[span_48](end_span).

4.3 Creation of Slides (Presentation)

[span_49](start_span)

Presentation software (like Microsoft PowerPoint or Google Slides) is used to create a sequence of slides to accompany a presentation[span_49](end_span).

Key Features and Concepts

    [span_50](start_span)
  • Creation of Slides: Adding new slides with different layouts (e.g., Title Slide, Title and Content)[span_50](end_span).
    • [span_51](start_span)
  • Inserting & Editing Text: Adding and modifying text inside text boxes on a slide[span_51](end_span).
    • [span_52](start_span)
  • Slide Transition: The visual effect that occurs when moving from one slide to the next during a presentation (e.g., Fade, Wipe)[span_52](end_span).
    • [span_53](start_span)
  • Animation: Visual effects applied to individual items on a slide, such as text, images, or charts (e.g., Fly In, Appear)[span_53](end_span).
