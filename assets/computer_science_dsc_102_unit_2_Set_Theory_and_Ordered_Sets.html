2.1 Basic Concept of Set Theory[span_7](end_span)

A set is a well-defined, unordered collection of distinct objects, called elements or members.
  • Roster Notation: $A = \{1, 2, 3, 4\}$
  • Set-Builder Notation: $B = \{x \mid x \text{ is an even integer}\}$
  • Empty Set: $\emptyset$ or \{\} (A set with no elements).
  • Universal Set (U): The set containing all possible elements under consideration.
  • Subset: $A \subseteq B$ ("A is a subset of B") if every element of A is also in B.
  • Proper Subset: $A \subset B$ ("A is a proper subset of B") if $A \subseteq B$ and $A \ne B$.
  • Power Set: $P(A)$ is the set of all subsets of A. If $|A| = n$, then $|P(A)| = 2^n$.
[span_8](start_span)

2.2 Operations with Sets[span_8](end_span)

  • Union: $A \cup B = \{x \mid x \in A \text{ or } x \in B\}$
  • Intersection: $A \cap B = \{x \mid x \in A \text{ and } x \in B\}$
  • Difference: $A - B = \{x \mid x \in A \text{ and } x \notin B\}$
  • Complement: $A' = \{x \mid x \in U \text{ and } x \notin A\}$ (This is $U - A$)
  • Principle of Inclusion-Exclusion (for two sets): $|A \cup B| = |A| + |B| - |A \cap B|$

2.3 Functions[span_9](end_span)

A function $f$ from a set $A$ (domain) to a set $B$ (codomain), written $f: A \to B$, is an assignment of exactly one element of $B$ to each element of $A$.
  • Range: The set of all actual output values $\{f(a) \mid a \in A\}$. The range is a subset of the codomain.
  • Injective (One-to-One): $f(a) = f(b)$ implies $a = b$. Different inputs map to different outputs.
  • Surjective (Onto): For every $y \in B$, there exists an $x \in A$ such that $f(x) = y$. The range equals the codomain.
  • Bijective (One-to-One and Onto): The function is both injective and surjective.

2.4 Relations[span_10](end_span)

A binary relation $R$ from $A$ to $B$ is a subset of the Cartesian Product $A \times B = \{(a, b) \mid a \in A, b \in B\}$. If $A=B$, we say $R$ is a relation on $A$.

Example: If $A = \{1, 2, 3\}$, $R = \{(a, b) \mid a < b\}$ would be $R = \{(1, 2), (1, 3), (2, 3)\}$.

2.5 Properties of Relations[span_11](end_span)

Let $R$ be a relation on a set $A$.

  • Reflexive: $(a, a) \in R$ for every $a \in A$.
  • Irreflexive: $(a, a) \notin R$ for every $a \in A$.
  • Symmetric: If $(a, b) \in R$, then $(b, a) \in R$.
  • Anti-symmetric: If $(a, b) \in R$ and $(b, a) \in R$, then $a = b$.
  • Transitive: If $(a, b) \in R$ and $(b, c) \in R$, then $(a, c) \in R$.
Equivalence Relation: A relation that is Reflexive, Symmetric, and Transitive. Partial Order: A relation that is Reflexive, Anti-symmetric, and Transitive.

2.6 Representing Relations[span_12](end_span)

  • Zero-One Matrix ($M_R$): Let $A=\{a_1, ..., a_m\}$ and $B=\{b_1, ..., b_n\}$. The $m \times n$ matrix $M_R$ is defined by: $M_R[i, j] = 1$ if $(a_i, b_j) \in R$, and $0$ otherwise.
    • Reflexive: Main diagonal is all 1s.
    • Symmetric: $M_R = M_R^T$ (Matrix is symmetric).
    • Anti-symmetric: If $M_R[i, j] = 1$ and $i \ne j$, then $M_R[j, i] = 0$.
  • Directed Graph (Digraph): Draw a dot (vertex) for each element in $A$. Draw a directed arrow from $a$ to $b$ if $(a, b) \in R$. A loop from $a$ to $a$ represents $(a, a) \in R$.

2.7 Composition of Relations[span_13](end_span)

Let $R$ be a relation from $A$ to $B$, and $S$ be a relation from $B$ to $C$. The composition $S \circ R$ is a relation from $A$ to $C$ defined as:

$S \circ R = \{(a, c) \mid \text{there exists some } b \in B \text{ such that } (a, b) \in R \text{ and } (b, c) \in S\}$

If using matrices, $M_{S \circ R} = M_R \cdot M_S$ (using Boolean matrix multiplication, where $1+1=1$).

Powers of a relation: $R^2 = R \circ R$, $R^3 = R^2 \circ R$, etc.

[span_14](start_span)

2.8 Closures of Relations[span_14](end_span)

The closure of a relation $R$ with respect to a property $P$ is the smallest relation $R^*$ that contains $R$ and has property $P$.

  • Reflexive Closure: $R \cup \Delta$, where $\Delta = \{(a, a) \mid a \in A\}$ is the diagonal relation.
  • Symmetric Closure: $R \cup R^{-1}$, where $R^{-1} = \{(b, a) \mid (a, b) \in R\}$ is the inverse relation.
  • Transitive Closure ($R^*$): $R^* = R \cup R^2 \cup R^3 \cup ... \cup R^n$ (where $n$ is the number of elements in the set). It can be found efficiently using Warshall's Algorithm.

2.9 Ordered Sets[span_15](end_span)

A set $S$ with a partial ordering relation $R$ (must be reflexive, anti-symmetric, transitive) is called a partially ordered set or poset, denoted $(S, R)$.

  • Total Order (Linear Order): A partial order where every pair of elements is comparable (i.e., for any $a, b \in S$, either $(a, b) \in R$ or $(b, a) \in R$).
  • Example: $(\mathbb{Z}, \le)$ is a total order. The "subset" relation $(\subseteq)$ on a power set $P(A)$ is a partial order, but not a total order (e.g., $\{a\}$ and $\{b\}$ are incomparable).

2.10 Hasse Diagrams of Partially Ordered Sets[span_16](end_span)

A Hasse diagram is a simplified way to draw a finite poset.

  1. Start with the digraph of the partial order.
  2. Remove all loops (reflexivity is assumed).
  3. Remove all edges that are implied by transitivity (e.g., if $a \to b$ and $b \to c$, remove the direct edge $a \to c$).
  4. Arrange all vertices so that all arrows point upwards, then remove the arrowheads (direction is implied to be up).

In a Hasse diagram, $a$ is "below" $b$ and connected by a line if $a$ is related to $b$ and there is no $c$ such that $a$ is related to $c$ and $c$ is related to $b$ (i.e., $b$ *covers* $a$).
