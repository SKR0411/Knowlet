7. Define height-balanced tree.

[span_21](start_span)

A height-balanced tree (such as an AVL tree) is a binary tree in which the height of the left and right subtrees of every node differs by no more than one[span_21](end_span).

16. (a) What is stack? Algorithms for stack operations. (b) Recursion and its advantages. [6+4=10]

[span_32](start_span)

Stack: A linear data structure that follows the LIFO (Last In First Out) principle[span_32](end_span).

Algorithms:

  • PUSH(item): If TOP == MAX-1, Overflow; else { TOP = TOP + 1; STACK[TOP] = item; [span_33](start_span)}
  • POP(): If TOP == -1, Underflow; else { item = STACK[TOP]; TOP = TOP - 1; return item; }[span_33](end_span)
[span_34](start_span)

Recursion: A process in which a function calls itself directly or indirectly[span_34](end_span).
[span_35](start_span)Advantages: Reduces code length, easier to solve problems like Tree traversals and Towers of Hanoi[span_35](end_span).

22. Algorithm for Binary Search and search steps for 45. [5+5=10]

Binary_Search(A, n, item):
    Set beg = 0, end = n-1
    While beg <= end:
        mid = (beg + end) / 2
        If A[mid] == item: return mid
        Else If item < A[mid]: end = mid - 1
        Else: beg = mid + 1
