Question 1

[2 Marks]

Define discrete random variable with examples. State two properties of random variable.

A random variable is discrete if it can take only a finite or countably infinite number of distinct values. Examples: Number of heads in 10 tosses of a coin, number of students in a class. Properties: 1. P(x_i) ≥ 0 for all i. 2. Σ P(x_i) = 1.

Question 4

[2 Marks]

Show that for a random variable X, E(X²) ≥ {E(X)}².

We know that Variance V(X) = E(X²) - {E(X)}². Since Variance is always non-negative (V(X) ≥ 0), it follows that E(X²) - {E(X)}² ≥ 0, hence E(X²) ≥ {E(X)}².

Question 6

[2 Marks]

Define mathematical expectation of discrete and continuous random variable. Prove E(aX + b) = aE(X) + b.

Expectation: For discrete X, E(X) = Σ x P(x). For continuous X, E(X) = ∫ x f(x) dx. Proof: E(aX + b) = ∫ (ax + b) f(x) dx = a ∫ x f(x) dx + b ∫ f(x) dx. Since ∫ f(x) dx = 1, we get aE(X) + b.

Question 11

[2 Marks]

What is hypergeometric distribution?

A discrete probability distribution that describes the probability of k successes in n draws, without replacement, from a finite population of size N that contains exactly K objects with a certain feature.

Question 14

[2 Marks]

Write the definition of continuous uniform distribution and obtain its mean.

f(x) = 1 / (b - a) for a ≤ x ≤ b, and 0 otherwise.

Mean: E(X) = ∫ [x / (b - a)] dx from a to b = [x² / 2(b - a)] from a to b = (b² - a²) / 2(b - a) = (a + b) / 2.

SECTION-B: Descriptive Answer Type

Answer any five questions. (5 x 10 = 50 Marks)

Question 16 (a)

[4 Marks]

Define probability density function (p.d.f.). Given f(x) = 6x(1 - x) for 0 ≤ x ≤ 1, find b such that P(X < b) = P(X > b).

p.d.f.: A function f(x) such that f(x) ≥ 0 and ∫ f(x) dx = 1 over the entire range.

Since P(X < b) = P(X > b) and the total probability is 1, P(X < b) must be 0.5. ∫ 6x(1-x) dx from 0 to b = 0.5. 6[x²/2 - x³/3] from 0 to b = 0.5 → 3b² - 2b³ = 0.5. By inspection, b = 0.5.

Question 18 (a)

[4 Marks]

Prove that E(X + Y) = E(X) + E(Y).

For continuous variables: E(X + Y) = ∫∫ (x + y) f(x,y) dx dy. = ∫∫ x f(x,y) dx dy + ∫∫ y f(x,y) dx dy. The first term is ∫ x [∫ f(x,y) dy] dx = ∫ x f_X(x) dx = E(X). Similarly, the second term is E(Y).

Question 20 (a)

[4 Marks]

Define cumulant-generating function. Discuss the effect of change of origin and scale.

CGF: K(t) = log M(t), where M(t) is the m.g.f.. Effect: Let Y = (X - a)/h. Then K_Y(t) = log E[exp(t(X-a)/h)] = -at/h + K_X(t/h). Only the first cumulant is affected by the change of origin, whereas all cumulants are affected by the change of scale.

Question 22 (a)

[3 Marks]

Obtain mean and variance of binomial distribution.

Let X ~ B(n, p). M(t) = (q + pe^t)^n. Mean μ'_1 = [d/dt M(t)] at t=0 = n(q + pe^t)^{n-1} (pe^t) at t=0 = np. Variance = μ'_2 - (μ'_1)² = [n(n-1)p² + np] - (np)² = npq.

Question 24 (a)

[4 Marks]

For Normal distribution, show that μ_2n = 1.3.5....(2n-1)σ^2n.

For N(μ, σ²), moments about mean are found using ∫ x^2n f(x) dx from -∞ to ∞. Using the recurrence relation μ_r = σ² (r-1) μ_{r-2}, we get μ_2 = σ², μ_4 = 3σ^4, and by induction, μ_2n = (2n-1)(2n-3)...1 σ^2n.

Exam Success Toolkit

Exam Tips

  • Always state the range of the random variable when defining a p.d.f. or distribution.
  • In MGF problems, remember that μ'_r is the r-th derivative evaluated at t=0.

Common Mistakes

  • Forgetting to check if the total probability ΣP(x) or ∫f(x) equals 1.
  • Confusing the Mean and Variance of different distributions (e.g., Gamma vs. Beta).

Important Formulas List

  • Binomial: Mean = np, Var = npq.
  • Poisson: Mean = Var = λ.
  • Normal: μ_2n = (2n-1)!! σ^2n.
