Question 1(a) [2+2=4]

Define random variable with example. [span_4](start_span)State the conditions for probability mass function (p.m.f.).[span_4](end_span)

Definition: A random variable is a real-valued function defined on the sample space of a random experiment. [span_5](start_span)It assigns a unique real number to every outcome in the sample space.[span_5](end_span)

[span_6](start_span)

Example: In tossing two fair coins, if X denotes the number of heads, then X can take values 0, 1, or 2.[span_6](end_span)

[span_7](start_span)

Conditions for p.m.f.: For a discrete random variable X with possible values x1, x2, ..., the function p(xi) is a p.m.f. if:[span_7](end_span)

    [span_8](start_span)
  1. p(xi) >= 0 for all i (Non-negativity).[span_8](end_span)
    2. [span_9](start_span)
  2. Sum over all i of p(xi) = 1 (Total probability is unity).[span_9](end_span)

Question 1(b) [4]

[span_10](start_span)

Define cumulative distribution function (c.d.f.) of a random variable for both discrete and continuous cases.[span_10](end_span)

[span_11](start_span)

The Cumulative Distribution Function F(x) is defined as the probability that the random variable X takes a value less than or equal to x: P(X <= x).[span_11](end_span)

    [span_12](start_span)
  • Discrete Case: F(x) = Sum of p(xi) for all xi <= x.[span_12](end_span)
    • [span_13](start_span)
  • Continuous Case: F(x) = Integral from -infinity to x of f(t) dt, where f(t) is the p.d.f.[span_13](end_span)

Question 1(c) [4]

[span_14](start_span)

Define conditional probability density function and joint distribution function of a two-dimensional random variable.[span_14](end_span)

Joint Distribution Function: F(x, y) = P(X <= x, Y <= y). [span_15](start_span)It represents the probability that X is less than or equal to x AND Y is less than or equal to y simultaneously.[span_15](end_span)

[span_16](start_span)

Conditional p.d.f.: The conditional p.d.f. of X given Y=y is defined as:[span_16](end_span)

f(x|y) = f(x, y) / f2(y), provided f2(y) > 0
[span_17](start_span)

Where f(x,y) is the joint p.d.f. and f2(y) is the marginal p.d.f. of Y.[span_17](end_span)

The joint p.d.f. of X and Y is f(x,y) = 4xy exp(-(x^2 + y^2)); x >= 0, y >= 0. Find marginal p.d.f. of X and Y. Are X and Y independent? [span_18](start_span)Find conditional distributions.[span_18](end_span)

Step 1: Marginal p.d.f. of X
[span_19](start_span)Integrate f(x,y) with respect to y from 0 to infinity:[span_19](end_span)
f1(x) = Integral[4xy exp(-x^2) exp(-y^2) dy]
f1(x) = 4x exp(-x^2) * Integral[y exp(-y^2) dy]
Let y^2 = t, 2y dy = dt. Integral becomes 1/2 * Integral[exp(-t) dt] = 1/2.
[span_20](start_span)f1(x) = 2x exp(-x^2); x >= 0[span_20](end_span)

Step 2: Marginal p.d.f. of Y
[span_21](start_span)By symmetry, f2(y) is calculated similarly:[span_21](end_span)
[span_22](start_span)f2(y) = 2y exp(-y^2); y >= 0[span_22](end_span)

Step 3: Independence
[span_23](start_span)X and Y are independent if f(x,y) = f1(x) * f2(y).[span_23](end_span)
(2x exp(-x^2)) * (2y exp(-y^2)) = 4xy exp(-(x^2+y^2)) = f(x,y).
[span_24](start_span)Conclusion: X and Y are independent.[span_24](end_span)

[span_25](start_span)

Step 4: Conditional Distributions[span_25](end_span)
f(x|y) = f(x,y)/f2(y) = f1(x) = 2x exp(-x^2)
[span_26](start_span)f(y|x) = f(x,y)/f1(x) = f2(y) = 2y exp(-y^2)[span_26](end_span)

Question 4(a)(i) [5]

Define Variance. [span_37](start_span)Prove V(aX + bY) = a^2 V(X) + b^2 V(Y) + 2ab Cov(X,Y).[span_37](end_span)

[span_38](start_span)

Definition: V(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2.[span_38](end_span)

Proof:
Let E(X) = ux and E(Y) = uy.
E(aX + bY) = a*ux + b*uy.
V(aX + bY) = E[((aX + bY) - (a*ux + b*uy))^2]
= E[(a(X-ux) + b(Y-uy))^2]
= E[a^2(X-ux)^2 + b^2(Y-uy)^2 + 2ab(X-ux)(Y-uy)]
= a^2 E(X-ux)^2 + b^2 E(Y-uy)^2 + 2ab E[(X-ux)(Y-uy)]
[span_39](start_span)V(aX + bY) = a^2 V(X) + b^2 V(Y) + 2ab Cov(X,Y)[span_39](end_span)

[span_40](start_span)

If independent: Cov(X,Y) = 0, so V(aX + bY) = a^2 V(X) + b^2 V(Y).[span_40](end_span)

Question 10(a)(i) [3]

[span_51](start_span)

Obtain the median of normal distribution.[span_51](end_span)

[span_52](start_span)

For a Normal distribution N(u, sigma^2), the p.d.f. is symmetric about x = u.[span_52](end_span)
By definition of median (M): Integral from -infinity to M of f(x) dx = 0.5.
Since the total area under the symmetric curve is 1 and it is divided equally at the mean (u), we have:
[span_53](start_span)Median = Mean = Mode = u[span_53](end_span)
