SECTION-B: Short Answer Type

Answer any five questions. (5 x 2 = 10 Marks)

Question 21

What is R? Write its two advantages.
R is a language and environment for statistical computing and graphics. Advantages: 1. It is free and open-source. 2. It has a vast ecosystem of packages for advanced data analysis.

Question 23

Define xlab, ylab, col, main with respect to data visualization in R.
These are graphical parameters used in plotting functions:

  • main: The main title of the plot.
  • xlab: Label for the x-axis.
  • ylab: Label for the y-axis.
  • col: The color used for the plotted points or lines.

Question 24

Define Boxplot with explaining all of its parts.
A boxplot (or whisker plot) displays the distribution of data based on a five-number summary: minimum, first quartile (Q1), median (Q2), third quartile (Q3), and maximum. The "box" represents the Interquartile Range (IQR), the line inside is the median, and "whiskers" extend to the minimum and maximum values (excluding outliers).

Question 27

Write a note on kurtosis explaining its different types.
Kurtosis measures the "tailedness" of a probability distribution. Types: 1. Mesokurtic (Normal distribution). 2. Leptokurtic (Sharper peak, heavy tails). 3. Platykurtic (Flatter peak, thin tails).

SECTION-C: Descriptive Answer Type

Answer any five questions. (5 x 5 = 25 Marks)

Question 31

Write a note on different operators in R.
R supports several types of operators:

  • Arithmetic Operators: +, -, *, /, ^ (exponent), %% (modulus).
  • Relational Operators: <, >, <=, >=, ==, !=.
  • Logical Operators: & (AND), | (OR), ! (NOT).
  • Assignment Operators: <- or =.

Question 33

Write down the R-code for drawing visualizations.
Assuming data vector x:

  • Bar Diagram: barplot(x)
  • Pie Diagram: pie(x)
  • Histogram: hist(x)
  • Frequency Polygon: plot(density(x)) or lines(hist(x))

Question 35

Write R-code to find statistical measures for data: 7, 3, 5, 9, 22, 7, 11, 13, 15, 10.
data <- c(7, 3, 5, 9, 22, 7, 11, 13, 15, 10)
m <- mean(data)
med <- median(data)
v <- var(data)
s <- sd(data)
se <- sd(data)/sqrt(length(data))

Question 40

Define simple and multiple linear regression and provide code for y = a + bx.
Simple Linear Regression: Models the relationship between one dependent variable (y) and one independent variable (x). Multiple Linear Regression: Models the relationship between one dependent variable and two or more independent variables.
R-code for y = a + bx:
y <- c(7, 5, 3, 2, 11)
x <- c(4, 2, 9, 3, 2)
model <- lm(y ~ x)
summary(model) # To find values of a (Intercept) and b (Slope)
