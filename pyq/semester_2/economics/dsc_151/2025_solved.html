Question 2 10 Marks

Option A

(i) Define limit of a function. [2 Marks]

A limit is the value that a function approaches as the input approaches some value. [span_8](start_span)Mathematically, lim x->a f(x) = L means f(x) can be made arbitrarily close to L by taking x sufficiently close to a.[span_8](end_span)

(ii) Find the limit: lim x->∞ [sqrt(x² + 1) - sqrt(x² - 1)]. [3 Marks]

Rationalizing the numerator:

  • Multiply by [sqrt(x² + 1) + sqrt(x² - 1)] / [sqrt(x² + 1) + sqrt(x² - 1)]
  • Numerator: (x² + 1) - (x² - 1) = 2
  • Denominator: sqrt(x² + 1) + sqrt(x² - 1)
  • As x -> ∞, the denominator approaches ∞.
    • [span_9](start_span)
  • Limit = 2 / ∞ = 0[span_9](end_span)

(iii) Find the limit: lim x->0 (e^x - 1) / x. [3 Marks]

Using L'Hopital's Rule (since it is 0/0 form):

  • Derivative of numerator: e^x
  • Derivative of denominator: 1
    • [span_10](start_span)
  • Limit x->0 (e^x / 1) = e^0 = 1[span_10](end_span)

(iv) Determine whether f(x) = (x² - 4) / (x - 2) is continuous at x = 2. [2 Marks]

  • f(2) = (4-4)/(2-2) = 0/0, which is undefined.
    • [span_11](start_span)
  • Since f(2) is not defined, the function is not continuous at x = 2.[span_11](end_span)

Question 6 10 Marks

Option B (i) Profit Maximization [span_26](start_span)

C = q³ - 10q² + 17q + 66, Price (P) = 5.[span_26](end_span)

  • Profit (π) = Total Revenue - Total Cost = 5q - (q³ - 10q² + 17q + 66)
  • π = -q³ + 10q² - 12q - 66
  • dπ/dq = -3q² + 20q - 12 = 0
  • Solving for q: (3q - 2)(q - 6) = 0 => q = 6 (Check 2nd order: -6q + 20 < 0 at q=6).
    • [span_27](start_span)
  • Profit is maximized at q = 6.[span_27](end_span)

Exam Tips

  • Always check the Second Order Condition (SOC) for Maxima/Minima questions.
  • In Matrix questions, double-check signs in cofactors.

Common Mistakes

  • Forgetting the constant 'C' in indefinite integrals.
  • Incorrectly applying the quotient rule in differentiation.

Important Formulas

  • Profit Max: MR = MC
  • Consumer Surplus: Integral(P dx) - P0*Q0
