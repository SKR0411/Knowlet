UNIT-I

Question 1 [1 × 3 = 3 Marks]

(a) Is symbolic logic deductive in nature?

[span_4](start_span)

Yes, symbolic logic is primarily deductive in nature[span_4](end_span).

(b) Are 'Symbolic Logic' and 'Mathematical Logic' synonymous?

[span_5](start_span)

Yes, they are generally considered synonymous as both use mathematical symbols and methods to represent logical forms[span_5](end_span).

(c) What are Ideograms?

Ideograms are symbols that represent ideas or concepts directly rather than the sounds of words (e.g., symbols like ∨, ·, ⊃).

(d) What is the symbol of Negative Function/Negation?

[span_6](start_span)

The symbol is the Tilde (~) or sometimes a bar over the variable[span_6](end_span).

UNIT-II

Question 4 [1 × 3 = 3 Marks]

(a) What is a statement form?

[span_15](start_span)

A statement form is a sequence of symbols containing statement variables such that when statements are substituted for the variables, the result is a statement[span_15](end_span).

(b) What is Tautology?

[span_16](start_span)

A tautology is a statement form that is true under all possible assignments of truth values to its variables[span_16](end_span).

(c) If p is true and q is false, then what will be the value of p ⊃ q?

[span_17](start_span)

The value will be False[span_17](end_span).

(d) When does an argument become invalid?

[span_18](start_span)

An argument is invalid if and only if it is possible for all its premises to be true while its conclusion is false[span_18](end_span).

[span_19](start_span)

(b) What is assumed in shorter truth table method?

[span_20](start_span)

We assume the argument is invalid by assigning True to premises and False to the conclusion to see if a contradiction arises[span_20](end_span).

(c) How many rows are required in shorter truth table method?

[span_21](start_span)

Only one row is required[span_21](end_span).

(d) "Indirect truth table method is also known as shorter truth-table method." Is it true?

[span_22](start_span)

Yes, it is true[span_22](end_span).
