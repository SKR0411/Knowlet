Define limit of a function at x = a.[span_6](end_span)

A function f(x) is said to have a limit L as x approaches a if for every epsilon > 0, there exists a delta > 0 such that |f(x) - L| < epsilon whenever 0 < |x - a| < delta.

[span_22](start_span)
