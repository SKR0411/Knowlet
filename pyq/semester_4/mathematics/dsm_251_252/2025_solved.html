UNIT I: First Order Differential Equations

Question 1 (a)

[2 Marks] [span_4](start_span)

Write one solution for dy/dx = 3x².[span_4](end_span)

Integrating both sides with respect to x:

y = ∫ 3x² dx = x³ + C

One specific solution (letting C = 0) is y = x³.

Question 1 (c)

Determine whether (2xy + y²)dx + (x² + 2xy)dy = 0 is exact.[span_5](end_span)

[span_6](start_span)

Here, M = 2xy + y² and N = x² + 2xy.[span_6](end_span)

    [span_7](start_span)

[span_9](start_span)

Since ∂M/∂y = ∂N/∂x, the differential equation is exact.[span_9](end_span)

Question 2 (b)

[5 Marks] [span_10](start_span)

Solve dy/dx + 2y/x = x³.[span_10](end_span)

[span_11](start_span)

This is a linear differential equation of the form dy/dx + Py = Q, where P = 2/x and Q = x³.[span_11](end_span)

    [span_12](start_span)
  1. Integrating Factor (IF): IF = e^(∫ P dx) = e^(∫ 2/x dx) = e^(2 ln x) = x².[span_12](end_span)
    2. [span_13](start_span)
  2. General Solution: y * (IF) = ∫ Q * (IF) dx + C[span_13](end_span)
    3. [span_14](start_span)
  3. y * x² = ∫ x³ * x² dx + C = ∫ x⁵ dx + C[span_14](end_span)
    4. [span_15](start_span)
  4. y * x² = x⁶/6 + C[span_15](end_span)
[span_16](start_span)

Final Answer: y = x⁴/6 + C/x².[span_16](end_span)

Question 4 (a)

[5 Marks] [span_19](start_span)

Find the general solution of y'' + 3y' + 2y = e⁻²ˣ.[span_19](end_span)

1. [span_20](start_span)Complementary Function (CF): Characteristic equation is m² + 3m + 2 = 0.[span_20](end_span)

    [span_21](start_span)
  • (m + 1)(m + 2) = 0 ⇒ m = -1, -2.[span_21](end_span)
    • [span_22](start_span)
  • CF = C₁e⁻ˣ + C₂e⁻²ˣ.[span_22](end_span)

2. [span_23](start_span)Particular Integral (PI): PI = [1 / (D² + 3D + 2)] e⁻²ˣ.[span_23](end_span)

[span_24](start_span)

Since D = -2 makes the denominator zero, we use the formula for a repeated root: PI = x * [1 / (2D + 3)] e⁻²ˣ = x * [1 / (2(-2) + 3)] e⁻²ˣ = -xe⁻²ˣ.[span_24](end_span)

[span_25](start_span)

General Solution: y = C₁e⁻ˣ + C₂e⁻²ˣ - xe⁻²ˣ.[span_25](end_span)

UNIT III: Cauchy-Euler & Variation of Parameters

Question 5 (b)

[2 Marks] [span_26](start_span)

What substitution reduces x²y'' - 4xy' + 6y = 0 into a constant coefficient equation?[span_26](end_span)

[span_27](start_span)

The standard substitution is x = eᶻ or z = ln x.[span_27](end_span)

[span_28](start_span)

The transformed equation becomes: [D(D-1) - 4D + 6]y = 0 ⇒ (D² - 5D + 6)y = 0, where D = d/dz.[span_28](end_span)

Question 6 (b)

[5 Marks] [span_29](start_span)

Solve x²y'' + 5xy' + 6y = 0 using substitution.[span_29](end_span)

[span_30](start_span)

Substitute x = eᶻ, then x(dy/dx) = Dy and x²(d²y/dx²) = D(D-1)y.[span_30](end_span)

[span_31](start_span)

Equation becomes: [D(D-1) + 5D + 6]y = 0 ⇒ (D² + 4D + 6)y = 0.[span_31](end_span)

[span_32](start_span)

Characteristic roots: m = [-4 ± √(16 - 24)] / 2 = -2 ± i√2.[span_32](end_span)

[span_33](start_span)

y = e⁻²ᶻ [C₁ cos(√2 z) + C₂ sin(√2 z)].[span_33](end_span)

[span_34](start_span)

Final Answer: y = x⁻² [C₁ cos(√2 ln x) + C₂ sin(√2 ln x)].[span_34](end_span)

  3. Using multipliers (dx-dy)/( (x-y)-(x+y) ) = dz/z leads to second characteristic C₂.[span_47](end_span)

General Solution: Φ(C₁, C₂) = 0.[span_48](end_span)
