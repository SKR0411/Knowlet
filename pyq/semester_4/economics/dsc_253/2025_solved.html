8. (b) (ii) Standard Error calculation. 3

[span_43](start_span)

Given: N = 41, n = 5, σ = 6.25. (Sampling without replacement)

SE = (σ / √n) * √[(N - n) / (N - 1)]

SE = (6.25 / √5) * √[(41 - 5) / (41 - 1)] = (6.25 / 2.236) * √(36/40) = 2.795 * 0.948 = 2.65[span_43](end_span)

[span_49](start_span)

    • [span_50](start_span)
  • Consistency: As sample size increases, the estimator should approach the population parameter.[span_50](end_span)
