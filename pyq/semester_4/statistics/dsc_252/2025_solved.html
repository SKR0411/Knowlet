Question 1 (a) [2 Marks]

[span_5](start_span)

Show that for a singleton set, supremum is equal to infimum.[span_5](end_span)

[span_6](start_span)

Let S = {a} be a singleton set containing one real number a.[span_6](end_span)

    [span_7](start_span)
  • Since 'a' is the only element, every element x in S satisfies x <= a and x >= a.[span_7](end_span)
  • By definition, the Supremum (Least Upper Bound) is the smallest value that is greater than or equal to all elements in the set. [span_8](start_span)Here, sup(S) = a.[span_8](end_span)
  • Similarly, the Infimum (Greatest Lower Bound) is the largest value less than or equal to all elements. [span_9](start_span)Here, inf(S) = a.[span_9](end_span)
    • [span_10](start_span)
  • Therefore, sup(S) = inf(S) = a.[span_10](end_span)

Find the supremum and infimum of the set S = {1/n; n in N}.[span_11](end_span)

[span_12](start_span)

The set is S = {1, 1/2, 1/3, 1/4, ...}.[span_12](end_span)

  • Supremum: The largest element in the set is 1 (when n=1). [span_13](start_span)Thus, sup(S) = 1.[span_13](end_span)
  • Infimum: As n approaches infinity, 1/n approaches 0. Although 0 is not in the set, it is the greatest lower bound. [span_14](start_span)Thus, inf(S) = 0.[span_14](end_span)

Define interior point and open set. [span_15](start_span)Show that finite intersection of open sets is also an open set.[span_15](end_span)

[span_16](start_span)Interior Point: A point 'x' is an interior point of set S if there exists a neighborhood (x - epsilon, x + epsilon) contained entirely within S.[span_16](end_span) [span_17](start_span)
Open Set: A set is open if every point in the set is an interior point.[span_17](end_span)

Proof: Let G1 and G2 be two open sets. [span_18](start_span)Let G = G1 intersection G2.[span_18](end_span)

    [span_19](start_span)
  1. If G is empty, it is open by definition.[span_19](end_span)
    2. [span_20](start_span)
  2. If x belongs to G, then x belongs to G1 and x belongs to G2.[span_20](end_span)
    3. [span_21](start_span)
  3. Since G1 is open, there is an epsilon1 such that (x - epsilon1, x + epsilon1) is in G1.[span_21](end_span)
    4. [span_22](start_span)
  4. Since G2 is open, there is an epsilon2 such that (x - epsilon2, x + epsilon2) is in G2.[span_22](end_span)
  5. Let epsilon = min(epsilon1, epsilon2). [span_23](start_span)Then the neighborhood (x - epsilon, x + epsilon) is contained in both G1 and G2, and thus in G.[span_23](end_span)
    6. [span_24](start_span)
  6. Hence, G is an open set.[span_24](end_span)

[span_40](start_span)Statement: If f(x) is continuous in [a, b], differentiable in (a, b), and f(a) = f(b), then there exists at least one 'c' in (a, b) such that f'(c) = 0.[span_40](end_span)
[span_41](start_span)

Proof:[span_41](end_span)

    [span_42](start_span)
  • If f(x) is constant, then f'(x) = 0 for all x in (a, b).[span_42](end_span)
    • [span_43](start_span)
  • If f(x) is not constant, since it's continuous on [a, b], it attains its maximum M and minimum m.[span_43](end_span)
    • [span_44](start_span)
  • Since f(a) = f(b), at least one of M or m must be attained at an interior point 'c'.[span_44](end_span)
    • [span_45](start_span)
  • If f(c) is a maximum, then f(c+h) <= f(c).[span_45](end_span)
    • [span_46](start_span)
  • This implies the right-hand derivative is <= 0 and left-hand derivative is >= 0.[span_46](end_span)
    • [span_47](start_span)
  • Since f'(c) exists, both must equal 0. Thus, f'(c) = 0.[span_47](end_span)
