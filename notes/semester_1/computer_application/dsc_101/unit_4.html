Basics of Computer Networks

[cite_start]

A computer network is a collection of computers and other hardware components interconnected by communication channels that allow sharing of resources and information[cite: 10, 24, 156, 171].

[cite_start]

Networks are categorized based on the geographical area they cover[cite: 24, 171].

Network Topology

[cite_start]

Network topology refers to the physical or logical arrangement of computers or nodes in a network[cite: 24, 171]. Common types include:

  • Bus Topology: All nodes are connected to a single central cable.
  • Star Topology: All nodes are connected to a central hub or switch.
  • Ring Topology: Each node is connected to exactly two other nodes, forming a circle.
  • Mesh Topology: Every node is connected to every other node.

Concept of Internet & WWW

    [cite_start]
  • Internet: A global system of interconnected computer networks that use the Internet protocol suite (TCP/IP) to link devices worldwide[cite: 10, 24, 156, 171].
    • [cite_start]
  • WWW (World Wide Web): A system of interlinked hypertext documents accessed via the Internet[cite: 24, 171].

IP Address, URL, and ISP

    [cite_start]
  • IP Address: A unique numerical label assigned to each device connected to a computer network[cite: 24, 171].
    • [cite_start]
  • Website Address and URL: Uniform Resource Locator (URL) is the global address of documents and other resources on the World Wide Web[cite: 24, 171].
    • [cite_start]
  • ISP (Internet Service Provider): An organization that provides services for accessing, using, or participating in the Internet[cite: 24, 171].

Modes of Connecting Internet

[cite_start]

There are several ways to connect a device to the Internet[cite: 24, 171]:

    [cite_start]
  • Hotspot: Using a mobile device's cellular data to create a local Wi-Fi network[cite: 24, 171].
    • [cite_start]
  • Wi-Fi: Wireless networking technology that allows devices to interface with the Internet[cite: 24, 171].
    • [cite_start]
  • LAN Cable: Physical ethernet connection for high-speed stability[cite: 24, 171].
    • [cite_start]
  • Broadband: High-speed Internet access that is always on[cite: 24, 171].
    • [cite_start]
  • USB Tethering: Connecting a phone to a computer via USB to share the phone's internet connection[cite: 24, 171].

Surfing the Web & Search Engines

[cite_start]

Exploring the Internet involves "surfing" through web pages using specialized tools[cite: 24, 171].

    [cite_start]
  • Search Engines: Tools used to search for information on the World Wide Web (e.g., Google, Bing)[cite: 24, 171].
    • [cite_start]
  • Downloading/Printing: Saving web pages for offline use or creating physical copies[cite: 24, 171].

Exam Focus & Tips

    [cite_start]
  • Exam Tip: Be ready to differentiate between the Internet and the World Wide Web; they are not the same thing[cite: 24, 171].
  • Common Mistake: Confusing URL with IP Address. [cite_start]A URL is the human-readable name (www.google.com), while an IP is the machine-readable number[cite: 24, 171].
    • [cite_start]
  • Mnemonic: LMW - Local (Small), Metropolitan (City), Wide (World)[cite: 24, 171].
