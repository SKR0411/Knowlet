Basic Concepts of Set Theory

[cite_start]

Set theory is the fundamental language of mathematics[cite: 73]. [cite_start]A set is a well-defined collection of distinct objects[cite: 73].

    [cite_start]
  • Elements: The objects belonging to a set are called its members or elements[cite: 73].
    • [cite_start]
  • Notation: Sets are usually denoted by capital letters (A, B, C) and elements by small letters (a, b, x)[cite: 73].
    • [cite_start]
  • Universal Set (U): The set containing all objects under consideration[cite: 73].
    • [cite_start]
  • Empty Set (Ø): A set with no elements[cite: 73].

Operations with Sets

[cite_start]

Common operations used to manipulate sets include[cite: 73]:

    [cite_start]
  • Union (A ∪ B): Elements in A, or B, or both[cite: 73].
    • [cite_start]
  • Intersection (A ∩ B): Elements common to both A and B[cite: 73].
    • [cite_start]
  • Difference (A - B): Elements in A that are not in B[cite: 73].
    • [cite_start]
  • Complement (A'): Elements in the Universal set not in A[cite: 73].
    • [cite_start]
  • Cartesian Product (A × B): The set of all ordered pairs (a, b) where a is in A and b is in B[cite: 73].

Functions

[cite_start]

A function is a special type of relation where every element in the domain is mapped to exactly one element in the codomain[cite: 73].

    [cite_start]
  • Domain: The set of all possible inputs[cite: 73].
    • [cite_start]
  • Range: The set of all actual outputs[cite: 73].
    • [cite_start]
  • One-to-One (Injective): Every element in the range has only one pre-image[cite: 73].
    • [cite_start]
  • Onto (Surjective): Every element in the codomain is an image of at least one element in the domain[cite: 73].

Relations and Properties

[cite_start]

A relation R from set A to B is a subset of A × B[cite: 73].

Properties of Relations

    [cite_start]
  • Reflexive: (a, a) is in R for every a in A[cite: 73].
    • [cite_start]
  • Symmetric: If (a, b) is in R, then (b, a) must be in R[cite: 73].
    • [cite_start]
  • Transitive: If (a, b) and (b, c) are in R, then (a, c) must be in R[cite: 73].
    • [cite_start]
  • Anti-symmetric: If (a, b) and (b, a) are in R, then a = b[cite: 73].

Composition and Closures of Relations

    [cite_start]
  • Representing Relations: Can be done via sets of pairs, matrices, or directed graphs (digraphs)[cite: 73].
    • [cite_start]
  • Composition: If R is a relation from A to B and S is from B to C, S ∘ R is a relation from A to C[cite: 73].
    • [cite_start]
  • Closures: The smallest relation containing R that satisfies a specific property (Reflexive, Symmetric, or Transitive closure)[cite: 73].

Ordered Sets and Hasse Diagrams

[cite_start]

A set with a partial ordering relation is called a Partially Ordered Set or Poset[cite: 73].

Hasse Diagrams

[cite_start]

A Hasse diagram is a simplified graphical representation of a finite partially ordered set[cite: 73].

    [cite_start]
  • Eliminates reflexive loops[cite: 73].
    • [cite_start]
  • Eliminates transitive edges[cite: 73].
    • [cite_start]
  • Direction is understood to be upwards[cite: 73].

Exam Focus & Tips

    [cite_start]
  • Exam Tip: Be prepared to draw a Hasse Diagram for a given set and relation (like "divides" or "subset of")[cite: 73].
  • Common Mistake: Confusing "Symmetric" and "Anti-symmetric". [cite_start]A relation can be neither, or even both (if it's empty)[cite: 73].
    • [cite_start]
  • Derivation: To find the Reflexive Closure, add (a, a) pairs to the existing relation until all elements are covered[cite: 73].

Frequently Asked Questions

Q: What is a Power Set?
A: The set of all possible subsets of a set, including the empty set and the set itself.

Q: When is a relation called an Equivalence Relation?
A: When it is Reflexive, Symmetric, and Transitive.
