Control Structures and Nested Loops

[cite_start]

Control structures determine the flow of execution based on conditions or repetitions[cite: 201].

Conditional Statements

    [cite_start]
  • IF-ELSE: Executes code based on whether a condition is true or false[cite: 201].
    • [cite_start]
  • SWITCH: Selects one of many code blocks to be executed based on a variable's value[cite: 201].

Looping Structures

    [cite_start]
  • WHILE: Repeats a block while a condition is true[cite: 201].
    • [cite_start]
  • DO-WHILE: Executes at least once, then repeats if the condition is true[cite: 201].
    • [cite_start]
  • FOR: Repeats a block for a specific number of times[cite: 201].
    • [cite_start]
  • Nested Loops: A loop inside another loop, often used for matrix operations[cite: 201].

Jump Statements

    [cite_start]
  • BREAK: Exits the current loop or switch[cite: 201].
    • [cite_start]
  • CONTINUE: Skips the rest of the current iteration and moves to the next one[cite: 201].
    • [cite_start]
  • GOTO: Transfers control to a labeled statement (use with caution)[cite: 201].

Frequently Asked Questions

Q: What is the difference between a WHILE and DO-WHILE loop?
[cite_start]A: A WHILE loop checks the condition first, whereas a DO-WHILE loop executes the body at least once before checking the condition[cite: 201].

Q: What does the % operator do?
[cite_start]A: It returns the remainder of an integer division[cite: 200].
