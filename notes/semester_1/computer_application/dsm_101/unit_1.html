Control Structures: Looping

[cite_start]

Loops are used to repeat a block of code multiple times[cite: 109].

  • WHILE Loop: Repeats as long as a condition is true. [cite_start]The condition is checked before execution[cite: 109].
    • [cite_start]
  • DO-WHILE Loop: Similar to while, but the condition is checked after execution, ensuring the loop runs at least once[cite: 109].
  • FOR Loop: Best used when the number of iterations is known. [cite_start]It combines initialization, condition, and increment/decrement[cite: 109].

Jump Statements

    [cite_start]
  • BREAK: Exits the current loop or switch immediately[cite: 109].
    • [cite_start]
  • CONTINUE: Skips the remaining code in the current iteration and jumps to the next iteration[cite: 109].
    • [cite_start]
  • GOTO: Transfers control to a labeled part of the program (use sparingly)[cite: 109].

Frequently Asked Questions

Q: What is the difference between = and ==?
A: = is for assignment; [cite_start]== is for checking equality[cite: 108].

Q: When is a DO-WHILE loop preferred over a WHILE loop?
[cite_start]A: When you need the code block to execute at least once regardless of the condition[cite: 109].
