Basics of Structures and Processing

[cite_start]

A structure is a user-defined data type in C that allows you to combine data items of different kinds[cite: 124]. [cite_start]Unlike arrays, which store data of the same type, structures can group variables like an int, a float, and a char together under a single name[cite: 124].

    [cite_start]
  • Definition: Created using the struct keyword followed by the structure name and a list of members inside curly braces[cite: 124].
    • [cite_start]
  • Processing: Individual members are accessed using the dot operator (.) for structure variables[cite: 124].

User Defined Data Types (typedef)

[cite_start]

The typedef keyword is used to provide a new name for an existing data type[cite: 124]. [cite_start]It is commonly used with structures to simplify their declaration in the code[cite: 124].

Example: typedef struct { int x; int y; [cite_start]} Point; allows you to declare a point as Point p1; instead of struct Point p1;[cite: 124].

Structures with Pointers and Functions

    [cite_start]
  • Structures and Pointers: You can create a pointer that points to the address of a structure[cite: 124]. [cite_start]When using pointers, members are accessed using the arrow operator (->)[cite: 124].
    • [cite_start]
  • Passing to Functions: Structures can be passed to functions by value (a copy is sent) or by reference (a pointer is sent)[cite: 124]. [cite_start]Passing by reference is more memory-efficient for large structures[cite: 124].

Arrays of Structures and Self-referential Structures

    [cite_start]
  • Arrays of Structures: Useful for storing a list of records, such as an array of 50 students where each student has a name and a roll number[cite: 124].
    • [cite_start]
  • Self-referential Structures: A structure that contains a pointer to a structure of the same type[cite: 124]. [cite_start]These are the building blocks of linked lists and trees[cite: 124].

Unions and Table Lookup

[cite_start]

A Union is similar to a structure, but all its members share the same memory location[cite: 124].

    [cite_start]
  • Memory Usage: A union's size is equal to the size of its largest member[cite: 124]. [cite_start]It can only hold the value of one member at a time[cite: 124].
    • [cite_start]
  • Table Lookup: Structures are often used to create lookup tables for storing and retrieving related data efficiently[cite: 124].

Exam Focus & Tips

  • Exam Tip: A common question is the difference between Structures and Unions. [cite_start]Focus on the memory sharing aspect of Unions[cite: 124].
    • [cite_start]
  • Common Mistake: Forgetting the semicolon (;) at the end of a structure definition[cite: 124].
  • Mnemonic: Structures = Separate memory; [cite_start]Unions = United (shared) memory[cite: 124].

Frequently Asked Questions

Q: What is a self-referential structure?
[cite_start]A: It is a structure that has a pointer as a member which points back to the same structure type[cite: 124].

Q: Why use typedef with structures?
[cite_start]A: To create a shorter, more readable alias for the structure type, making the code cleaner[cite: 124].
