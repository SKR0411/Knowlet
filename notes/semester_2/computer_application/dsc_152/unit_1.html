2. Core Concept: C++ Templates

[cite_start]The syllabus explicitly requires the use of Template functions for searching and sorting tasks[cite: 74, 75].

Definition: Templates allow a function or class to work with different data types (int, float, char) without rewriting the code for each type.

Example Template Structure:

template <typename T>
int linearSearch(T arr[], int n, T key) {
  for(int i = 0; i < n; i++) {
    if(arr[i] == key) return i;
  }
  return -1;
}

3. Task 1: Linear and Binary Search

[cite_start]Assignment: Write a program to search an element from a list, giving the user the option to perform Linear or Binary search using template functions[cite: 73, 74].

Implementation Details

    [cite_start]
  • Linear Search: Scans every element[cite: 27]. Ideal for unsorted lists.
    • [cite_start]
  • Binary Search: Requires a sorted list[cite: 27]. Uses the divide-and-conquer approach to locate elements in O(log n) time.

4. Task 2: Insertion, Bubble, and Selection Sort

Assignment: WAP using templates to sort a list. [cite_start]Give users the option to choose between Insertion, Bubble, or Selection sort[cite: 75, 76].

Logic Summaries for Coding

[cite_start] [cite_start] [cite_start]
Algorithm Key Logic C++ Concept
Bubble SortCompare adjacent elements and swap if needed[cite: 27]. Nested loops with swap().
Selection SortFind the minimum element and swap it with the first unsorted element[cite: 27]. Tracking min_index.
Insertion SortBuild the sorted list one element at a time by inserting it into position[cite: 27]. Shifting elements.

5. Lab Exam Tips & Troubleshooting

Exam Focus: In practical exams, ensure your menu-driven interface is clear and handles invalid inputs gracefully.
    [cite_start]
