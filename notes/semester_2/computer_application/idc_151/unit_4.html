1. Decision Making and Branching

[cite_start]Decision making structures require that the programmer specify one or more conditions to be evaluated or tested by the program[cite: 306]. [cite_start]Branching allows the program to skip or execute certain blocks of code based on the result of these tests[cite: 306].

2. The IF Family

[cite_start]The if statement is the most powerful tool for controlling the flow of execution[cite: 306].

    [cite_start]
  • Simple IF Statement: Used to execute a single block of code if a condition is true[cite: 306].
    • [cite_start]
  • IF-ELSE Statement: Executes one block if the condition is true and another block if it is false[cite: 306].
    • [cite_start]
  • Nested IF-ELSE: An if-else statement inside another if or else block, used for complex multi-level decisions[cite: 306].
    • [cite_start]
  • ELSE IF Ladder: A chain of if statements where multiple conditions are checked sequentially[cite: 306].

3. Switch Statement

[cite_start]The switch statement is a multi-way branch statement[cite: 306]. [cite_start]It provides an easy way to dispatch execution to different parts of code based on the value of an expression[cite: 306].

[cite_start]Utility: It is often used as a cleaner alternative to a long else if ladder when comparing a single variable against multiple constant values[cite: 306].

4. Looping: Introduction and Types

[cite_start]Looping involves repeating a set of instructions until a specific condition is met[cite: 307]. [cite_start]This is essential for tasks like processing array elements or generating series[cite: 307].

5. While, Do-While, and For Statements

[cite_start]There are three primary looping constructs in C[cite: 307]:

    [cite_start]
  • while Statement: An entry-controlled loop that checks the condition before executing the loop body[cite: 307].
    • [cite_start]
  • do-while Statement: An exit-controlled loop that executes the body at least once before checking the condition[cite: 307].
    • [cite_start]
  • for Statement: A compact loop structure that includes initialization, condition, and increment/decrement in a single line[cite: 307].

Exam Tips

    [cite_start]
  • Semicolon Warning: Never put a semicolon after the if(condition) or for(...) header, as it terminates the statement immediately without executing the block[cite: 306, 307].
    • [cite_start]
  • Switch Break: Always remember to use the break keyword in switch cases; otherwise, execution will "fall through" to subsequent cases[cite: 306].
    • [cite_start]
  • Loop Selection: Use a for loop when you know the number of iterations in advance and a while loop when the termination depends on a condition[cite: 307].

Frequently Asked Questions

Q1: What is the main difference between while and do-while?
[cite_start]The while loop may never execute if the condition is false initially, whereas do-while guarantees at least one execution[cite: 307].

Q2: When should I use an ELSE IF ladder over a Switch?
[cite_start]Use an else if ladder for range-based conditions (e.g., marks > 90) and a switch for specific discrete values (e.g., choice == 1)[cite: 306].
