1. Character Set and C Tokens

[cite_start]The Character Set includes all characters that the C compiler can recognize, such as letters (A-Z, a-z), digits (0-9), and special characters[cite: 303].

[cite_start]C Tokens are the smallest individual units in a program[cite: 303]. These include:

    [cite_start]
  • Keywords: Reserved words with predefined meanings[cite: 303].
    • [cite_start]
  • Identifiers: Names given to program elements[cite: 303].
    • [cite_start]
  • Constants: Fixed values that do not change[cite: 303].
    • [cite_start]
  • Operators: Symbols for mathematical or logical operations[cite: 303].

2. Keywords and Identifiers

[cite_start]Keywords are reserved words that have a special meaning to the compiler and cannot be used as variable names (e.g., int, float, if)[cite: 303].

[cite_start]Identifiers are names used for variables, functions, and arrays[cite: 303].

    [cite_start]
  • They must start with a letter or an underscore[cite: 303].
    • [cite_start]
  • Only letters, digits, and underscores are allowed[cite: 303].
    • [cite_start]
  • They are case-sensitive[cite: 303].

3. Constants and Variables

[cite_start]Constants are values that remain unchanged during program execution[cite: 303]. [cite_start]These can be numeric (integer, real) or character constants[cite: 303].

[cite_start]Variables are identifiers used to store data values that can change during execution[cite: 303].

    [cite_start]
  • Declaration: Variables must be declared before use, specifying their data type[cite: 303].
    • [cite_start]
  • Assigning Values: Values are assigned using the assignment operator (=)[cite: 303].
    • [cite_start]
  • Symbolic Constants: Defined using the #define preprocessor to assign a name to a constant value[cite: 303].

4. Data Types and Declarations

[cite_start]Data Types specify the type of value a variable can store[cite: 303].

    [cite_start]
  • Primary Data Types: int (integer), float (real), char (character), and double (double-precision real)[cite: 303].
    • [cite_start]
  • Declaration: All variables must be declared to inform the compiler about the storage space required[cite: 303].

5. Operators and Expressions

[cite_start]Operators are symbols that trigger an action when applied to variables and constants[cite: 304].

Arithmetic +, -, *, /, %Perform basic math like addition and modulus[cite: 304].

Frequently Asked Questions

Q1: What is a keyword?
[cite_start]A keyword is a word reserved by the C language for specific purposes (like int for integers) and cannot be used as a name for variables[cite: 303].

Q2: What is the difference between a constant and a variable?
[cite_start]A constant's value is fixed and cannot be changed during the program, whereas a variable acts as a container whose content can be modified[cite: 303].
