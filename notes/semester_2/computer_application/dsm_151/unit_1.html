4. Control Structures

[cite_start]Control structures manage the flow of execution within a program[cite: 155].

Selection Statements

    [cite_start]
  • IF-ELSE: Executes a block of code if a condition is true; otherwise, it executes another block[cite: 155].
    • [cite_start]
  • SWITCH: A multi-way branch statement that selects one of many code blocks to be executed based on a variable's value[cite: 155].

Iterative Statements (Loops)

    [cite_start]
  • WHILE: Repeats a block of code as long as a condition is true[cite: 155].
    • [cite_start]
  • DO-WHILE: Similar to while, but the block is executed at least once before the condition is checked[cite: 155].
    • [cite_start]
  • FOR: Used for a fixed number of iterations, often using a counter[cite: 155].

Jump Statements

    [cite_start]
  • BREAK: Terminates the current loop or switch statement immediately[cite: 155].
    • [cite_start]
  • CONTINUE: Skips the remaining code in the current iteration and moves to the next loop iteration[cite: 155].
    • [cite_start]
  • GOTO: Transfers control to a labeled statement elsewhere in the function[cite: 155].

Exam Tips

  • Syntax: Always remember that C is case-sensitive. "Int" is not the same as "int".
    • [cite_start]
  • Semicolons: A very common mistake is forgetting the semicolon (;) at the end of statements[cite: 154].
    • [cite_start]
  • Loops: Ensure every loop has a clear exit condition to avoid "Infinite Loops"[cite: 155].

Mnemonic: WDF

To remember the three main types of loops in C: While, Do-While, For.

Frequently Asked Questions

Q1: What is the difference between WHILE and DO-WHILE?
The while loop checks the condition first and may never run. [cite_start]The do-while loop runs at least once because it checks the condition at the end[cite: 155].

Q2: Why is the GOTO statement generally discouraged?
[cite_start]It makes programs harder to read, debug, and maintain by creating "spaghetti code" that jumps unpredictably[cite: 155].
