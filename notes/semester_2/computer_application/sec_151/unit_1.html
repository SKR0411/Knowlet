1. Basic Elements of Python

Python is a high-level, interpreted programming language known for its readability and simple syntax. [cite_start]The basic elements include character sets and the use of comments to explain code[cite: 220].

  • Comments: Used to add notes within the code. [cite_start]Python uses the # symbol for single-line comments[cite: 220].

[cite_start]A token is the smallest individual unit in a python program[cite: 220].

    [cite_start]
  • Keywords: Reserved words that have special meaning to the interpreter (e.g., if, else, while)[cite: 220].
    • [cite_start]
  • Identifiers: Names given to entities like variables, functions, and classes[cite: 220].
    • [cite_start]
  • Literals: Constant values stored in variables (e.g., 10, "Hello")[cite: 220].
    • [cite_start]
  • Operators: Symbols that perform operations on variables and values[cite: 220].
    • [cite_start]
  • Punctuators: Symbols used to organize sentence structures and expressions (e.g., , : [ ] { })[cite: 220].

3. Variables, L-values, and R-values

[cite_start]Variables are reserved memory locations to store values[cite: 220].

    [cite_start]
  • L-value: Expressions that refer to a memory location (e.g., a variable on the left side of an assignment)[cite: 220].
    • [cite_start]
  • R-value: The data value stored at some address in memory (e.g., the value/expression on the right side of an assignment)[cite: 220].

5. Mutable vs Immutable Types

[cite_start]Understanding mutability is critical for managing data in Python[cite: 220].

    [cite_start]
  • Mutable Types: Values can be changed after they are created (e.g., Lists, Dictionaries, Sets)[cite: 220].
    • [cite_start]
  • Immutable Types: Values cannot be changed once created (e.g., Numbers, Strings, Tuples)[cite: 220].
