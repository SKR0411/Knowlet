3. Exception Handling (Try/Except)

[cite_start]Exceptions are errors detected during execution that interrupt the normal flow of the program[cite: 215, 236].

    [cite_start]
  • Try Block: Contains the code that might raise an exception[cite: 236].
    • [cite_start]
  • Except Block: Handles the error if one occurs in the try block[cite: 236].
    • [cite_start]
  • Finally Block: Contains code that will execute regardless of whether an exception was raised or not, often used for cleanup[cite: 236].
    • [cite_start]
  • Built-in Exceptions: Python includes standard errors like ZeroDivisionError, FileNotFoundError, and ValueError[cite: 236].
