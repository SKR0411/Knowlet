1. Lists

[cite_start]A List is an ordered, mutable collection of items that can be of different data types[cite: 220, 226].

    [cite_start]
  • Indexing: Lists use zero-based indexing to access elements[cite: 226].
    • [cite_start]
  • Operations: Supports concatenation (joining lists), repetition (multiplying elements), membership (checking if item exists), and slicing (extracting parts of a list)[cite: 226].
    • [cite_start]
  • Traversal: Lists can be traversed using loops to process each element sequentially[cite: 226].
    • [cite_start]
  • Nested Lists: Lists can contain other lists as elements[cite: 226].
    • [cite_start]
  • Built-in Methods: Python provides various methods to manipulate lists, such as append(), extend(), and insert()[cite: 226].

2. Tuples

[cite_start]A Tuple is an ordered, immutable sequence of items[cite: 220, 227]. [cite_start]Once created, its elements cannot be changed[cite: 220].

    [cite_start]
  • Indexing: Similar to lists, tuples use indexing to access specific items[cite: 227].
    • [cite_start]
  • Operations: Supports concatenation, repetition, membership, and slicing[cite: 227].
    • [cite_start]
  • Built-in Functions: Common functions include len(), max(), and min()[cite: 227].

3. Sets

[cite_start]A Set is an unordered collection of unique elements[cite: 213, 228].

    [cite_start]
  • Creation: Sets are created using curly braces {} or the set() function[cite: 228].
    • [cite_start]
  • Modification: Elements can be added or removed, but sets do not support indexing or slicing because they are unordered[cite: 228].
    • [cite_start]
  • Operations: Includes mathematical set operations like union, intersection, difference, and symmetric difference[cite: 228].
    • [cite_start]
  • Methods: Common methods include add(), remove(), and pop()[cite: 228].

4. Dictionaries

[cite_start]A Dictionary is a mapping of unique keys to values[cite: 220, 229].

    [cite_start]
  • Accessing Items: Values are accessed using their associated keys rather than index numbers[cite: 229].
    • [cite_start]
  • Mutability: Dictionaries are mutable, allowing for adding new terms or modifying existing items[cite: 229].
    • [cite_start]
  • Traversal: You can traverse a dictionary to access keys, values, or key-value pairs using loops[cite: 229].
    • [cite_start]
  • Built-in Methods: Includes keys(), values(), and items()[cite: 229].

5. Python Modules

[cite_start]A Module is a file containing Python definitions and statements that can be reused in other programs[cite: 230].

    [cite_start]
  • Importing: Modules are imported using the import <module> statement or the from ... import ... syntax[cite: 230].
    • [cite_start]
  • Math Module: A built-in module providing access to mathematical functions like sqrt(), sin(), and cos()[cite: 230].

6. String Manipulation

[cite_start]Advanced string handling techniques are vital for data processing[cite: 231].

    [cite_start]
  • Basic Operations: Includes string concatenation and repetition[cite: 231].
    • [cite_start]
  • Slicing: Using range indices to extract substrings[cite: 231].

Common Mistakes

  • Trying to change a tuple element (Tuples are immutable!)[cite: 220].
    • [cite_start]

    • [cite_start]
  • Assuming sets maintain the order in which items were added[cite: 228].

Frequently Asked Questions

Q1: When should I use a Tuple instead of a List?
[cite_start]Use a tuple when you have data that should not change throughout the program (e.g., coordinates, constant settings) to ensure data integrity[cite: 220].

Q2: How do I remove duplicates from a list?
[cite_start]The easiest way is to convert the list to a set using set(my_list) and then back to a list, as sets automatically handle unique elements[cite: 228].
